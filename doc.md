# version 1.0

```
git init #本地仓库
git add doc.md #添加文件到仓库
git commit -m "commit doc.md" #提交文件
```

# version 2.0

```
git status #查看状态
git add filename #提交修改
git restore filename #discard changes
git diff #查看修改
```

# version 3.0

```
git reset --hard commit_id #回退版本， HEAD,HEAD^,HEAD^^,HEAD~100
git log # commit log
git reflog # command log
```
